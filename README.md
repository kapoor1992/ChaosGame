# ChaosGame
Fractal maker using Turtle.

Inspired by [this](https://www.youtube.com/watch?v=kbKtFN71Lfs) video.

    Usage: 

    python chaos.py <shape, triangle/square/pentagon/hexagon> 
                    <points, int> 
                    <refesh rate, int> 
                    <halves, int> 
                    <no repeats, yes/no>
